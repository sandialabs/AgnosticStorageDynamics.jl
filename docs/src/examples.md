# Examples

This package uses a Literate workflow for executable examples.

- Main dynamic scenario (mirrors integration tests): [Dynamic Profile Example](generated/dynamic_profile.md)

The generated example includes:

1. fill to capacity and spill
2. idle standing losses
3. pull down to empty with unmet demand
4. random dynamic behavior
5. appended windows that stay below and exceed charge/discharge rate limits
6. optional annotated plotting of scenario windows
7. option-based mass/volume estimates with time-resolved system mass output
